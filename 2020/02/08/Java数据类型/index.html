1.基本数据类型

1.1整数型

整数型 占用字节 范围
byte 1 -128 ~ +127
short 2 -32768 ~ +32767
int 4 -2^31 ~ (+2^31)-1
long 8 -2^63 ~ (+2^63)-1

#默认为int型。

1.2浮点数型

浮点数型 占用字节 范围
float 4 -2^128 ~ +2^128
double 8 -2^1024 ~ +2^1024

#默认为double型。

1.3字符型

字符型 占用字节 范围
chart 2 0 ~ 65535

1.4布尔型

布尔型 占用字节 范围
boolean 1 ture/false
